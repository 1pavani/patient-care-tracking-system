# Patient Care Tracking System

## Overview

A full-stack healthcare management application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

* JWT Authentication
* Role-Based Access Control
* Patient Management
* Appointment Management
* Treatment Management
* Follow-Up Management
* Doctor Management
* Protected Routes
* Responsive Dashboard

## Technologies Used

### Frontend

* React.js
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## User Roles

### Admin

* Manage Doctors
* Manage Patients
* Manage Appointments
* Manage Treatments
* Manage Follow Ups

### Doctor

* View Appointments
* Manage Treatments
* Manage Follow Ups

### Receptionist

* Manage Patients
* Manage Appointments
